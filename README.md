# 🧠 Smart RAM Cleaner Pro

Умный очиститель оперативной памяти с защитой SSD, мониторингом в реальном времени и автозаморозкой вкладок Chrome.

## ✨ Возможности

- 🧹 **11 методов очистки ОЗУ** с защитой SSD
- 📊 **Мониторинг в реальном времени** (обновление каждые 500мс)
- 🌐 **Chrome-расширение** для заморозки неактивных вкладок
- 🖱️ **Клик колесиком** по иконке трея = мгновенная очистка
- 🔄 **Автоматические обновления** через GitHub Releases
- 🛡️ **Защита SSD** от лишних циклов записи
- 📈 **Детальная диагностика** утечек памяти

## 🚀 Установка

### Способ 1: Готовый EXE (рекомендуется)
1. Перейди в [Releases](https://github.com/DENLOPER/SmartRAMCleaner/releases)
2. Скачай последний `SmartRAMCleaner.exe`
3. Запусти от имени администратора

### Способ 2: Из исходников
```bash
git clone https://github.com/DENLOPER/SmartRAMCleaner.git
cd SmartRAMCleaner
pip install -r requirements.txt
python smart_ram_cleaner.py
